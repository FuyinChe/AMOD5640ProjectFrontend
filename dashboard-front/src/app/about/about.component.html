Project Goals

Our objective is to build a web-based tool for visualizing environmental sample data, including snow depth, humidity, temperature, and more, collected from Trent Climate Station (TCS) and Trent School of the Environment (TSE) Farm Station.

This project demonstrates the application of data science, web technologies, and interactive charts using Angular and Chart.js.
